# Pied-Piper

This is a simple blog web Application that allows users to create, read posts. And also comment on it. It is built with ReactJs and Firebase.

## Getting Started

To get started, clone this repository and install the dependencies:

```
git clone https://github.com/0rigin-c0de/Pied-Piper01.git

cd Pied-iper01

yarn

```

Then, start the development server:

```
yarn start
```

The app will be available at http://localhost:3000.

## Features

This app includes the following features:

- Users can create, read posts.
- Users can comment on posts.
- Posts are stored in Firebase.
- Images are also stored in firebase.
- Responsive to all types of screens
